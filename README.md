# LibraryManagementAPI

Library Management API that supports the calls listed below:

POST: /signup

POST: /users/login

POST: /book

GET: /books

GET: /book/{id}

PATCH: /book

DELETE: /book/{id}

POST: /author

GET: /authors

GET: /author/{id}

DELETE: /author/{id}

POST: /series

GET: /series

GET: /series/{id}

DELETE: /series/{id}

POST: /authors/{id}/books

GET: /authors/{id}/books

DELETE: /authors/{id}/books

POST: /series/{id}/books

GET: /series/{id}/books

DELETE: /series/{id}/books

GET: /books/{id}/author

GET: /books/{id}/series
